## Build Night — Guida

Anche a distanza, possiamo organizzare sessioni di "assemblaggio virtuale": ognuno lavora sul proprio PC mentre è in voice chat su Discord, condividendo lo schermo o chiedendo consigli in tempo reale.

### Cosa ti serve per partecipare a una Build Night online

| Cosa | Note |
|------|------|
| Componenti principali | Scheda madre, CPU, RAM, alimentatore, case, GPU (se hai) |
| Cacciaviti | A croce e a taglio, magnetici se possibile |
| Cavi in più | Cavi SATA, prolunghe, fascette |
| Unità di storage | SSD o HDD da installare |
| Connessione Discord | Per chiedere aiuto in diretta |

Se non hai componenti ma vuoi solo osservare e imparare, sei il benvenuto: condividiamo lo schermo e spieghiamo passo passo.

## Raspberry Pi

Il Raspberry Pi è un mini computer economico e versatile. Perfetto per imparare, sperimentare e creare progetti permanenti.

### Progetti consigliati

| Progetto | Difficoltà | Cosa serve |
|----------|------------|------------|
| Pi-hole | Facile | Blocca pubblicità a livello di rete |
| RetroPie | Facile | Emulatore per giochi retrò |
| Server domestico | Media | Condividi file in rete (NAS leggero) |
| Stazione meteo | Media | Sensori di temperatura/umidità |
| Telecamera IP | Media | Telecamera di sorveglianza fai-da-te |
| Cluster Raspberry Pi | Avanzata | Più Pi che lavorano insieme |

### Risorse utili

- **Raspberry Pi Imager** — strumento ufficiale per scrivere OS sulla SD
- **Raspberry Pi OS** — la distribuzione ufficiale, basata su Debian
- **Pinout.xyz** — schema dei pin GPIO

## Arduino ed Elettronica

Arduino è una piattaforma per progetti di elettronica programmabile. Perfetta per prototipi, sensori, automazioni e piccoli gadget.

### Primi passi

1. **Scegli un Arduino** — Uno, Nano, Mega (i più comuni)
2. **Installa l'IDE** — Arduino IDE o VS Code con estensione PlatformIO
3. **Impara le basi** — LED, pulsanti, resistenze
4. **Sperimenta con sensori** — temperatura, distanza, movimento

### Progetti semplici per iniziare

| Progetto | Cosa impari |
|----------|-------------|
| LED lampeggiante | Setup base, digitalWrite, delay |
| Termometro | Sensore analogico, seriale |
| Serratura con tastierino | Input digitale, controllo servo |
| Sensore di parcheggio | Sensore ultrasonico, buzzer |

## NAS fai da te (Home Server)

Un NAS (Network Attached Storage) è un server per conservare file in rete, accessibili da tutti i dispositivi di casa.

### Con cosa costruirlo

- **Hardware riciclato** — un vecchio PC con qualche hard disk
- **Raspberry Pi** — con disco esterno USB
- **Software open source**:
  - **OpenMediaVault** — facile, interfaccia web
  - **TrueNAS SCALE** — più potente, basato su Debian
  - **Rockstor** — basato su openSUSE

### Cosa puoi farci

- Backup automatici di PC e smartphone
- Server multimediale (Jellyfin / Plex)
- Condivisione file tra membri della famiglia
- Download automatizzati
